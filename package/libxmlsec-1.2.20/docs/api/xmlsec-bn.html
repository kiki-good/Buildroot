Description

Big numbers support functions.

Details

enum xmlSecBnFormat

typedef enum {
    xmlSecBnBase64,
    xmlSecBnHex,
    xmlSecBnDec
} xmlSecBnFormat;

The big numbers formats.

xmlSecBnBase64

the base64 decoded binary blob.
xmlSecBnHex

the hex number.
xmlSecBnDec

the decimal number.
