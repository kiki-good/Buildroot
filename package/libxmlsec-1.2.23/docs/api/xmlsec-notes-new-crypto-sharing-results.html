Sharing the results.

If you implemented support for new cryptographic library (or extended an existing one) and both you and your company/university/... are willing to share the code I would be glad to add your work to XML Security Library. Many people will thank you for this and will use your library. Of course, you'll get all the credits for your work.

The best way to submit your enchancements is to send a pull request through GitHub. I will try to review and merge your pool request as soon as possible.
